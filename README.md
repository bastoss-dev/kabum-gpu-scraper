# Kabum GPU Scraper

Script em Python que coleta automaticamente os preços de placas de vídeo da Kabum usando web scraping.

## 🚀 Tecnologias utilizadas

- Python
- Requests
- BeautifulSoup
- CSV

## 📦 O que o script faz

- coleta nome das placas de vídeo
- coleta preços
- salva os dados em um arquivo CSV

## ▶️ Como executar

1. Instale as dependências

pip install requests beautifulsoup4

2. Execute o script

python scraper.py

## 📁 Saída

O script gera um arquivo:

placas_de_video.csv

contendo os produtos e seus preços.
